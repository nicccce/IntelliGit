# Views — 业务视图

此目录存放按业务场景划分的页面级视图组件。

## 规划视图

| 视图           | 说明                               |
| -------------- | ---------------------------------- |
| `Workspace/`   | 工作区视图（文件状态、暂存、提交） |
| `History/`     | 历史拓扑图（Git Log 可视化）       |
| `Settings/`    | 应用设置页面                       |
| `BranchGraph/` | 分支合并拓扑图                     |

每个视图应为独立文件夹，至少包含 `index.tsx`；需要局部样式时，再在视图目录内增加 `styles.css`。

## 当前视图

| 视图            | 说明                          |
| --------------- | ----------------------------- |
| `ChangesView/`  | 变更、暂存、Diff 和提交       |
| `HistoryView/`  | 分支、Commit Graph 和提交详情 |
